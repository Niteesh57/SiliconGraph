STEP 3

Build an optimized package

The package stores this device's capacity-aware CPU/Vulkan policy. At runtime it selects only a graph and KV-cache size that fit current free RAM. Vulkan is used only after its benchmark beats CPU; below 50% battery it uses direct, concise answers without changing validated precision. For media-capable models, image/audio/video are admitted only on FP32/FP16 or INT8/mixed graphs; INT4/FP8 packages remain text-only.
